# Golden Minute

> **Be the vital link.** Let's build a future where a medical emergency is met with immediate action from the community.

## About the Project
**Golden Minute** is an innovative first-responder platform designed to combat sudden cardiac arrest (SCA). A victim's chances of survival decrease by 7-10% for every minute that passes without cardiopulmonary resuscitation (CPR) and defibrillation (AED).

Our application reduces the critical response time by instantly alerting verified volunteers within the most efficient travel radius around the victim, providing navigation instructions, and locating the nearest automated external defibrillators (AEDs).

## Key Features
* **Instant Alerts:** Notifies nearby volunteers the moment an emergency is triggered.
* **Smart Routing:** Guides the volunteer to the exact incident location in the shortest possible time.
* **AED Network:** Identifies and maps local automated external defibrillators for rapid intervention.
* **Scaling Survival:** The app's primary goal is to increase the probability of survival to 80% by covering the critical first minutes until the ambulance arrives.

## Technologies Used
The project is built on a modern Client-Server architecture:
* **Frontend (Mobile App):** React Native / Expo 
* **Backend (API):** Java / Spring Boot
* **Database & Infrastructure:** * **Docker:** Containerized database environment for managing the AED network data.
    * **Firebase:** Cloud services and real-time integration.
* **Backend Build Tool:** Maven
